This is a demo site for the Neoclassical Hugo theme. It is intended to show off the features and design of the theme, and to provide a starting point for users who want to create their own website with Neoclassical. The site includes a variety of content types, including blog posts, pages, and media, to showcase the different features and capabilities of the Neoclassical theme. 

Here are a few highlights of the Neoclassical theme:

- Banner image above the menu can be set on a per-page basis, with a site-wide default. It can use site assets (as on this page) or page resources (as seen [here](/article/europa-clipper-radiation-readiness)). You can use any standard raster image formats that Hugo supports (jpg, png, gif, webp) as well as SVG vector images.
- Sidebar modules can also be configured on a per-page basis, with site-wide defaults. This feature was used to remove the Recent and Featured posts module on the home page of this site.
- The base template includes pre-content and post-content blocks that will render any partial set via params. This allows for unique content to be added above and below the main content on a per-page basis, as seen on the home page of this site.
